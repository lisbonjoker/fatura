# Política de Segurança

## Reportar uma Vulnerabilidade

Para reportar uma vulnerabilidade de segurança, por favor [abra uma issue](https://github.com/lisbonjoker/invoice/issues/new) no GitHub com o título a começar por `[SEGURANÇA]`.

Inclua na issue:
- Descrição da vulnerabilidade
- Passos para reproduzir
- Impacto potencial

Todas as reports serão revistas e respondidas o mais rapidamente possível.
